2017-06-10 - 模拟电影投票网页做界面- 用bootstrap和jquery做 www.gewara.com （手机页面看）模仿做这个app 参考资料： www.bootcss.com （查看各种class用法） 1.首页 2.下拉菜单 3.页面切换 4.底部浮动（始终在底部） 5.电影和影院界面 1. 新建项目文件夹 2. 文件夹下- 右键git bash - "git init" - 出现隐藏文件夹“.git” 3. npm init 4. bower init - 生成两个json文件 5. bower install bootstrap --save (bower bootstrap#* ...就是在安装) *（其实这个生成的bootstrap和jquery是重点，本项目即依赖这两个） *重点文件：bower_components\bootstrap\dist\css\bootstrap.min.css bower_components\bootstrap\dist\js\bootstrap.min.js bower_components\jquery\dist\jquery.min.js 6. 新建文件夹1和文件夹DIST，在1中新建script和style文件夹，新建index.html 7. 先gulp -v 看是否存在， 再npm install gulp --save-dev（有点慢，如果不行就先全局安装） 8. 新建gulpfile.js 9. 新建README.md 10. git远程连接 - 最外层文件夹下- git bash-"touch .gitignore" - 生成一个空文件 （这个无严格顺序） 11. git推送到远程 12. 在index.html文件中写， 注意先引入三个重要文件， 参考bootstrap官网用上各种class来设样式 13.在gitignore文件中写以下三行：（表示不上传这些） bower_components node_modules myWork/img 14. 总文件夹下， git bash- "ssh git@github.com" - 成功后， 去网页登录自己的github - 新建项目 - 复制ssh地址 15. git bash- "git remote add origin (+刚才复制的地址)" git add . git commit -m "second commit" git push -u origin master -- 成功后， 即可看到自己的github网页上出现自己写的文件 *别人访问你的文件： git init git clone (+你的地址)
